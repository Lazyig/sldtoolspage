검색

페이지 최 상단의 텍스트 박스에 유닛명을 기입하고 엔터를 누르면 검색 됩니다.

유닛명 규칙

1. 유닛명의 앞뒤에는 언더바(_)를 붙인다.
  ex) 아몬
2. 유닛명은 띄워쓰기를 생략한다.
  ex) 비밀 작전 노바
3. 일부 유닛명은 축약한다.
  ex) 고르곤 전투 순양함, 원시 히드라리스크
